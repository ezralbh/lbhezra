Hubungkan ke GitHub

Masukkan Personal Access Token GitHub Anda untuk mengelola konten website secara langsung.

Cara mendapatkan token:
1. Buka GitHub Settings → Personal Access Tokens
2. Klik "Generate new token" (Fine-grained)
3. Pilih repository johnshean.github.io
4. Berikan izin Contents: Read and Write
5. Generate dan paste token di atas
